# Target Catch Game (8086 Assembly)

## Overview
This project is a simple game developed in 8086 Assembly using emu8086. 
Targets fall from the top of the screen, and the player moves left/right to catch them before they hit the ground.

## Features
- Player movement (left/right)
- Falling targets
- Collision detection
- Score system
- Game over condition
- Increasing difficulty

## Controls
- Left Arrow → Move Left
- Right Arrow → Move Right
- ESC → Exit Game

## Technologies Used
- 8086 Assembly Language
- emu8086
