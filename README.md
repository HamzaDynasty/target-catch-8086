# 🎮 COAL Target Catch Game

A simple DOS Assembly game made for the COAL (Computer Organization and Assembly Language) semester project.

## 📌 About
This game is built in x86 Assembly Language using DOS interrupts.  
The player controls the character `A` and catches falling objects to increase score.

## ✨ Features
- Left and right movement
- Falling targets
- Score system
- Collision detection
- Retro DOS-style UI
- Keyboard controls

## 🎮 Controls
| Key | Action |
|------|--------|
| ← | Move Left |
| → | Move Right |
| ESC | Exit Game |

## 🛠️ Technologies Used
- x86 Assembly Language
- DOS Interrupts
- EMU8086 / DOSBox
